Test Synopsis:
This test has three parts which run in sequence automatically:

  1. Initial test of the global metrology system (free drift for ~10 seconds)
  2. Open loop thruster firings to check thrusters, listen for even spacing (~3 seconds)
  3. Closed loop reorientation to check thrusters and estimation together (~13 seconds)
